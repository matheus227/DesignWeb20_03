-boas práticas de listas-
  1. Usar listas sempre que houver agrupamento de informações
  2. Não usar a tag br(tag de quebrar linhas) para simular listas o li já quebra linhas
  3. usar ol(lista ordenada) apenas quando a ordem for relevante
  4. usar ul para listas gerais
  5. usar dl para termos e definições (não está nesse documento pois é pouco usado)
  6. NÃO USA LI COMO SUBSTITUTO DO BR PARA QUEBRAR LINHAS
