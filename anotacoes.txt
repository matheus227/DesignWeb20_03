        ul = significa lista não ordenada, cada item da lista é representado por <li>
            lista não ordenada é vista em varios sites, por exemplo: menus de mavegação;
            categorias; características de um produto; tópicoss de um conteúdo
                itens não precsam seguir uma sequência.
        li = significa secamente falando item da lista

lista ordenada é utilizada quando a ordem dos itens é importante
        por  os itens são númerados (1 ,2 3...), mas podem ser configurados para serem letras ou
        algaarismos romanos        
        
        a lista ordenada é criada com a tag <ol> mas também utiliza o <li> para definir os itens
            para alteraar o tipo da ordem, se utiliza mais uma tag dentro da tag <ol>
                    <ol type="1"></ol> -- ordem numérica 
                    <ol type="A"></ol> -- ordem alfabética maiuscula
                    <ol type="a"></ol> -- ordem alfabética minuscula
                    <ol type="I"></ol> -- ordem algarismos romanos maisculos
                    <ol type="i"></ol> -- ordem algarismos romanos minusculos
